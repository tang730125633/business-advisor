### Skill 核心规则
1. **六步咨询工作流**：情绪识别 → 诊断 → 召回书籍 → 应用思维习惯 → 落地行业 → 克制输出
2. **五条硬规则**：300 字长度 / 情绪优先 / 颜文字多样 / 逐字引用 / 称呼锁定
3. **反幻觉铁律**：已知/未知事实清单 + 条件表达代替编造
4. **边界情形处理**：时效性陷阱、主观决策、情绪爆表、群消息识别
