# 《营销管理》- 菲利普·科特勒
> 分类：经营战略 | 关键词：STP 4P 顾客价值 市场导向 营销管理过程 客户关系 整合营销

## 一、核心思想（100字内）
营销不是促销也不是广告，而是一整套创造、传递、沟通顾客价值的管理过程。企业的中心任务是识别目标市场、理解客户需求、设计并传递超越对手的价值主张，最终建立长期盈利的客户关系。

## 二、核心框架/模型

### 框架1：STP（战略层营销）
- **Segmentation 市场细分**：按人口、地理、心理、行为划分市场
- **Targeting 目标选择**：评估细分市场的规模、成长性、吸引力、企业契合度
- **Positioning 市场定位**：在目标客户心中建立差异化的价值位置

### 框架2：4P / 4C（执行层营销组合）
- **Product 产品**：核心利益、实体产品、附加服务
- **Price 价格**：成本加成、价值定价、竞争定价
- **Place 渠道**：覆盖、中间商、物流、可得性
- **Promotion 促销**：广告、销售促进、公关、人员推销
- 对应客户视角的4C：Customer Value / Cost / Convenience / Communication

### 框架3：顾客让渡价值
- 顾客让渡价值 = 顾客总价值 − 顾客总成本
- 总价值包括：产品、服务、人员、形象
- 总成本包括：货币、时间、精力、心理
- 营销的本质是最大化这个差值

### 框架4：市场导向的五种哲学
- 生产观念 → 产品观念 → 推销观念 → 营销观念 → 社会营销观念
- 大多数企业还停留在推销观念，真正的营销观念是"从客户需求出发"

### 框架5：客户关系管理与客户终身价值
- 开发新客户的成本是保留老客户的5倍
- 重点在提升客户终身价值、忠诚度、复购率
- 建立客户数据库与分层服务

### 框架6：营销管理过程
- 分析市场机会 → 选择目标市场 → 设计营销战略 → 规划营销项目 → 组织实施与控制

## 三、决策触发场景

- **情境A**：老板觉得"产品很好，卖不动是销售的问题"
  → 应用"营销观念 vs 推销观念"
  → 思考方向：问题可能在上游——客户根本不是你设想的那群人，或者价值主张没戳中需求

- **情境B**：想同时抓大客户和小客户
  → 应用STP
  → 思考方向：这是两个完全不同的细分市场，需要两套价值主张，不能用一套打法

- **情境C**：产品同质化严重，只能打价格战
  → 应用"顾客让渡价值"
  → 思考方向：除了降价，还能从服务、响应速度、品牌形象、降低客户使用成本入手

- **情境D**：新老客户维护成本高，老板想砍客服预算
  → 应用"客户终身价值"
  → 思考方向：算一笔账——流失一个老客户要多少新客户才能补回来

- **情境E**：要进入一个新地区/新行业
  → 应用"营销管理过程"
  → 思考方向：先分析市场机会再谈产品，而不是带着现成产品去找需求

- **情境F**：营销预算不知道该怎么花
  → 应用4P均衡 + 目标倒推
  → 思考方向：四个P必须配合，广告不是唯一杠杆，渠道、价格、产品都可能是更关键的投入点

## 四、电力行业映射

- **场景：戴总要把综合能源服务卖给工商业客户**
  建议：应用STP——先细分（制造业/园区/商业综合体/数据中心），评估每类客户的节能需求与付费能力，再定位（例如"为中型制造业客户提供一站式用能优化"），而不是向所有客户推销同一套方案

- **场景：客户只看价格，销售总是在降价成交**
  建议：应用顾客让渡价值——将服务响应时间、系统稳定性、运维托底、投融资方案打包呈现，让客户看到总价值而非单价

- **场景：公司有大量存量项目业主，但每次都像做一锤子买卖**
  建议：应用客户关系管理——建立业主数据库，定期回访，从运维、扩容、改造、碳资产管理里挖掘终身价值
