# 《好战略，坏战略》- 理查德·鲁梅尔特
> 分类：经营战略 | 关键词：战略核心 调查分析 指导方针 连贯性行动 着力点 扬长避短 坏战略陷阱

## 一、核心思想（100字内）
好战略不是宏大口号，而是清晰认识挑战、找到关键着力点、集中资源施以连贯性行动。坏战略回避细节，用愿景、抱负和目标堆砌假装成战略。好战略有一个可诊断的核心结构，任何管理者都能学会。

## 二、核心框架/模型

### 框架1：战略核心的三要素（The Kernel）
- **调查分析**：诊断当前处境，识别最关键的障碍和机会
- **指导方针**：面对这些障碍的总体方针（路标，不是路线）
- **连贯性行动**：一系列相互协调、彼此强化的具体行动

### 框架2：坏战略的四个特征（警示信号）
- **空话连篇**：用复杂词汇装点空洞内容
- **回避问题**：不敢定义真正的挑战
- **误把目标当战略**：只喊"市场份额翻倍"没有方法
- **糟糕的战略目标**：目标过多、彼此冲突、不切实际

### 框架3：找到"着力点"（Leverage Point）
- 在众多问题中识别出"事半功倍"的那个
- 把有限资源压到这一个点上
- 打破敌人（或困境）的连贯性

### 框架4：扬长避短原则
- 战略不是弥补短板，而是设计一个让优势最大、劣势最小的战场
- 例子：鸣梁海峡（狭窄水道限制数量优势）、自己做深度长文（避开追热点能力短板）

### 框架5：连贯性行动的力量
- 所有行动必须相互强化，而不是彼此抵消
- 资源、人、流程都指向同一方向
- 连贯性本身就是竞争力

## 三、决策触发场景

- **情境A**：战略会议开完留下一堆漂亮PPT，但没人知道具体做什么
  → 应用坏战略识别
  → 思考方向：对照四个坏战略特征，大概率属于"误把目标/愿景当战略"

- **情境B**：面对强大对手，老板想在多个战场同时发力
  → 应用"着力点"原则
  → 思考方向：集中资源打一个最关键的点，而不是平摊

- **情境C**：公司短板明显，老板想花大力气补课
  → 应用"扬长避短"
  → 思考方向：不要补短板，去设计一个让长板决定胜负的战场

- **情境D**：各部门战略方向各自为政甚至相互冲突
  → 应用"连贯性行动"
  → 思考方向：所有动作是否指向同一指导方针？不指向的要砍

- **情境E**：战略执行不下去，高层指责执行力差
  → 应用战略核心三要素
  → 思考方向：问题往往不是执行，而是战略本身没有具体可行的连贯性行动

- **情境F**：制定年度目标时只是拍脑袋"增长20%"
  → 应用"好战略起于诊断"
  → 思考方向：先回答"当前最主要的障碍是什么"，再说目标

## 四、电力行业映射

- **场景：电力企业面临多重挑战（双碳、电改、竞争、融资），不知从何下手**
  建议：用战略核心三要素——先做诊断找出真正的"卡脖子"障碍（可能是负债率而非新能源），再针对这一关键问题制定指导方针和行动包

- **场景：新能源板块要与国有大集团竞争，老板想正面拼规模**
  建议：应用"扬长避短"——国企的短板是决策慢、机制僵；戴总的长板是灵活、区域深耕。选择需要快速响应的细分市场（如工商业分布式、园区综合能源），让规模变得没那么重要

- **场景：集团战略规划写成"做大做强做优"这类套话**
  建议：这是典型坏战略，应替换为一个明确的诊断和着力点，例如"未来三年的核心障碍是现金流紧张，一切动作围绕提升应收回款速度"
