# 《竞争战略》- 迈克尔·波特
> 分类：经营战略 | 关键词：五力模型 通用战略 总成本领先 差异化 集中战略 进入壁垒 行业结构 竞争对手分析

## 一、核心思想（100字内）
企业利润的高低主要由所处行业的结构性吸引力和企业在行业内的相对竞争地位决定。波特提供了一套系统工具：用五力模型分析行业结构，用三种通用战略选择竞争地位，用价值链定位建立可持续的竞争优势。

## 二、核心框架/模型

### 框架1：五力模型（行业结构分析）
- **现有竞争者之间的对抗强度**：集中度、增速、差异化程度、退出壁垒
- **新进入者的威胁**：取决于进入壁垒
- **替代品的威胁**：来自其他行业解决相同需求的产品
- **买方议价能力**：集中度、切换成本、对利润敏感度
- **供方议价能力**：集中度、替代供应、对买方重要性
- 五力决定行业平均利润率

### 框架2：进入壁垒的七大来源
- 规模经济
- 产品差异化（品牌忠诚）
- 资本要求
- 转换成本
- 分销渠道控制
- 与规模无关的成本劣势（专利、区位、经验曲线）
- 政府政策

### 框架3：三种通用竞争战略
- **总成本领先**：全行业范围内做到最低成本
- **差异化**：在客户看重的某些维度做到独特
- **集中战略**：聚焦某个细分市场，在其中追求成本或差异化
- 必须选一种——否则"进退两难"（Stuck in the Middle），两头不讨好

### 框架4：竞争对手分析四要素
- 未来目标
- 当前战略
- 对自身与行业的假设
- 能力
- 目的：预判对手的下一步和反应强度

### 框架5：市场信号与行动/反行动
- 发布会、价格变动、产能投资都是信号
- 读懂对手信号、发出自己的信号，避免无谓正面冲突

### 框架6：行业演变分析
- 分散型、新兴、成熟、衰退、全球化——每个阶段有不同的制胜逻辑
- 战略必须适应行业生命周期阶段

## 三、决策触发场景

- **情境A**：老板考虑进入一个新行业/新细分市场
  → 应用五力模型 + 进入壁垒分析
  → 思考方向：先看这个行业结构性盈利能力，再看自己有没有突破壁垒的独特资源

- **情境B**：利润率持续下降，不知道是自己的问题还是行业的问题
  → 应用五力模型
  → 思考方向：若整个行业的五力都在恶化，再努力也难逆转，该考虑转型或退出

- **情境C**：公司战略既想便宜又想高端，客户定位混乱
  → 应用"进退两难"警示
  → 思考方向：必须在三种通用战略里明确选一种

- **情境D**：被几个核心供应商或核心客户卡住利润空间
  → 应用"买方/供方议价力"
  → 思考方向：通过纵向一体化、多元采购、建立自有渠道等削弱对方议价力

- **情境E**：行业进入成熟期，增速放缓，竞争更激烈
  → 应用"成熟期转型"章节
  → 思考方向：从增长抢份额转向效率、从产品竞争转向服务和客户关系

- **情境F**：老板要判断某个对手的下一步
  → 应用"竞争对手分析四要素"
  → 思考方向：不只看对手在做什么，还要推断它的目标和假设

- **情境G**：行业内同类公司一大堆，大家都赚不到钱
  → 应用"分散型行业"策略
  → 思考方向：通过整合收购、建立集中化分销、差异化品牌来"非分散化"

## 四、电力行业映射

- **场景：分析电力行业某细分领域（如售电、EPC、运维、储能）是否值得进入**
  建议：完整跑一遍五力——替代品（自发自用、虚拟电厂）、买方（大工业客户议价强）、供方（电网/设备厂商集中）、进入壁垒（资质、牌照、资金）、对手对抗强度。高壁垒+低替代品的细分才是好战场

- **场景：面对电网央企与其他民营EPC双面夹击**
  建议：必须在通用战略中选一个——不要试图既低价又高端。可选"区域集中战略"（某省/某市做深做透）或"客户集中战略"（只服务某类工商业用户）

- **场景：综合能源服务/储能行业处于新兴期**
  建议：应用"新兴行业的竞争战略"——此阶段重点不是规模，而是树立行业标准、建立客户信任、锁定关键资源（优质站址、融资渠道、关键设备）
