# 《终身成长》- 卡罗尔·德韦克
> 分类：领导力 | 关键词：成长型思维、固定型思维、努力的意义、反馈方式、人才培养、能力可塑、失败观、夸奖的陷阱

## 一、核心思想（100字内）
人分两种思维模式：固定型认为能力天生、害怕失败以维持人设；成长型相信能力可通过努力提升、拥抱挑战。组织的思维模式决定了它能否留住人才、跨越危机。领导者最大的杠杆不是找天才，而是营造成长型文化，让普通人也能持续变强。

## 二、核心框架/模型

### 框架1：两种思维模式对比
| 维度 | 固定型思维 | 成长型思维 |
|---|---|---|
| 对能力的看法 | 天生固定 | 可通过练习发展 |
| 对挑战的态度 | 回避，怕暴露缺陷 | 拥抱，视为学习机会 |
| 对努力的态度 | 努力=没天赋 | 努力=通向精通之路 |
| 对批评的反应 | 防御、抗拒 | 倾听、吸收 |
| 对他人成功 | 感到威胁 | 汲取灵感 |
| 失败的意义 | 定义"我是谁" | 一次数据点 |

### 框架2：成长型组织的四个特征
- 招聘：看潜力和学习能力，不唯简历与过往头衔
- 反馈：夸奖努力和策略，而非"你真聪明"
- 允许失败：区分"探索型失败"和"执行型失败"
- 高层示范：CEO自曝短板与学习过程

### 框架3：错误的夸奖 vs 正确的夸奖
- 错误："你真聪明""你真有天赋"→ 孩子和下属会害怕失去这个标签，开始规避挑战
- 正确："你用了一个很好的方法""你坚持下来了""你从错误里学到了什么"→ 强化可控因素

### 框架4：固定型思维的CEO综合征
- 艾柯卡式：把公司当成自己伟大的纪念碑
- 邓拉普式：一切都是为了证明"我是超级巨星"
- 行为特征：拒绝听坏消息、打压能干下属、只想要光环项目、避免接班人培养

### 框架5：成长型CEO的行为
- 韦尔奇式：倾听、信任、培养，不怕承认错误
- 郭士纳式：接手IBM时先承认"我不懂"，然后学习
- 把公司看成一个学习型组织，而非自己的舞台

## 三、决策触发场景
- 情境A：年轻技术员频繁犯错、团队想淘汰
  → 判断是固定型还是成长型失败。如果对方在反思和调整，继续给机会；如果在掩饰和找借口，才处理
- 情境B：核心骨干说"我就这样了，再学新东西太累"
  → 典型固定型自我设限。不是能力问题，是思维模式。用小挑战和小成功重启成长曲线
- 情境C：公司要表彰先进员工
  → 不要颁"最聪明奖"，要颁"进步最大奖""攻坚克难奖"，否则会强化固定型文化
- 情境D：有高管抗拒数字化转型
  → 害怕新领域暴露无能。给他学习的时间与面子，让他先做"学习者"再做"决策者"
- 情境E：面试选接班人
  → 不要找"像领导的人"，找"会问问题、会承认不懂的人"
- 情境F：自己作为老板最近焦虑，怕被超越
  → 自己掉进了固定型陷阱，这是最大的风险

## 四、电力行业映射

**应用1：老中青三代的思维模式管理**
电力企业老师傅群体容易有固定型思维倾向："我干了三十年，还需要你教？"——这不是经验问题，是自我保护。建议戴总给老师傅设计"带徒弟"的荣誉角色，让他们的价值不再依赖"我什么都懂"，而是"我能教出多少徒弟"。这把固定型的标签转化为成长型的贡献。

**应用2：安全培训的反馈方式**
电力行业安全最怕"隐瞒小错"。如果公司文化是"出错就挨批"，员工会藏错。改成成长型反馈："这次差点出事，你是怎么发现的？下次怎么能更早发现？"——把事故和近因事件变成学习资产。一线班组的安全文化会快速改善。

**应用3：新业务（新能源、储能、综合能源服务）团队建设**
电力行业正从传统向新能源转型，这时候最需要成长型人才。老员工里的"我不懂就不做"是最大阻力。招聘转型团队时看两条：是否愿意承认"我不知道"，是否能讲出最近三个月学了什么新东西。这比看专业背景更可靠。
