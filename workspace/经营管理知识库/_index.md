## 一、知识库结构（24本书 / 5大栏目）

### 01 现代企业管理（管理方法论）· 5本
- 《高效能人士的七个习惯》- 柯维：个人+组织效能的底层操作系统
- 《卓有成效的管理者》- 德鲁克：管理者的5个核心习惯
- 《管理的实践》- 德鲁克：企业存在的目的=创造顾客
- 《原则》- 达利欧：可信度加权 + 极度透明的决策文化
- 《第五项修炼》- 圣吉：学习型组织 + 系统思考

### 02 经营战略 · 7本
- 《从优秀到卓越》- 柯林斯：飞轮效应 + 刺猬理念 + 第五级领导力
- 《创新者的窘境》- 克里斯坦森：破坏性创新 + 资源依赖
- 《基业长青》- 柯林斯：造钟 vs 报时 + 核心理念
- 《好战略，坏战略》- 鲁梅尔特：诊断-指导方针-行动
- 《定位》- 特劳特/里斯：抢占心智
- 《竞争战略》- 波特：五力模型 + 三大通用战略
- 《营销管理》- 科特勒：STP + 4P + 客户终身价值

### 03 营销（总经理视角）· 4本
- 《22条商规》- 特劳特/里斯：22条不可违背的营销定律
- 《影响力》- 西奥迪尼：说服的6大原则
- 《流量池》- 杨飞：品牌即流量池 + AARRR + 裂变
- 《超级转化率》- 陈勇：转化六要素

### 04 领导力·团队·组织 · 5本
- 《思考，快与慢》- 卡尼曼：系统1/系统2 + 认知偏差
- 《终身成长》- 德韦克：成长型思维 vs 固定型思维
- 《领导力21法则》- 麦克斯维尔：21条领导力定律
- 《首先，打破一切常规》- 白金汉：盖洛普Q12 + 优势管理
- 《高绩效团队》- 亨曼：高绩效团队的4C模型

### 05 商业思维·格局类 · 3本
- 《价值》- 张磊：长期主义 + 动态护城河
- 《穷查理宝典》- 芒格：多元思维模型 + 人类误判心理学
- 《苏世民：我的经验与教训》：25条工作与生活原则

## 二、跨书关联地图（按问题类型）

### 📊 战略决策类问题
**当戴总问"要不要进入/退出某个业务"时**：
- 主框架：《好战略，坏战略》的诊断-指导方针-行动
- 补充视角：《竞争战略》五力分析、《创新者的窘境》识别破坏性信号、《定位》心智占位、《价值》长期价值判断
- 风险校验：《穷查理宝典》反过来想、《思考快与慢》规避过度自信

### 👥 团队与人的问题
**当戴总问"团队不行、骨干要走、下属不主动"时**：
- 主框架：《首先，打破一切常规》因才适用 + 盖洛普Q12
- 补充：《高绩效团队》4C模型、《终身成长》成长型思维、《领导力21法则》
- 底层：《高效能人士的七个习惯》习惯4-6（互赖期）

### 🏗️ 组织与文化类问题
**当戴总问"企业文化怎么建、制度落不下去"时**：
- 主框架：《基业长青》造钟 + 核心理念
- 补充：《原则》极度透明决策、《第五项修炼》学习型组织
- 支撑：《管理的实践》德鲁克的管理原则

### 💰 营销与客户类问题
**当戴总问"新业务怎么打开市场、招投标策略"时**：
- 主框架：《定位》+《22条商规》
- 战术：《超级转化率》六要素、《影响力》6原则、《流量池》裂变
- 战略校准：《营销管理》STP

### 🎯 个人决策与格局问题
**当戴总问"要不要冒险、怎么看长期、接班"时**：
- 主框架：《穷查理宝典》多元思维模型
- 补充：《价值》长期主义、《苏世民》25原则、《原则》达利欧决策树
- 执行：《高效能人士的七个习惯》习惯1-3

### ⚡ 创新与转型类问题（电力行业核心！）
**当戴总问"新能源冲击、储能要不要做、数字化转型"时**：

## 三、针对戴总电力行业的通用映射原则

1. **电力行业是典型的重资产、长周期、强监管行业** → 优先调用长期主义（《价值》）、造钟思维（《基业长青》）、反脆弱（《穷查理宝典》）
2. **国企/民企混合，政府关系与市场化并存** → 《影响力》权威+社会认同、《22条商规》牺牲定律
3. **技术型团队 + 老中青三代传承** → 《首先打破一切常规》优势管理、《领导力21法则》
4. **新能源/储能/虚拟电厂对传统业务的冲击** → 《创新者的窘境》是第一优先
5. **招投标 + 工程项目制特点** → 《超级转化率》标书六要素结构
6. **安全文化与事故决策** → 《思考快与慢》系统2、卡尼曼偏差

## 四、OpenClaw 对话应用规则

当 qmd 召回本知识库的内容时，OpenClaw 应：

1. **不要直接引用书名说教**，而是自然地融入建议
   - ❌ "根据德鲁克在《卓有成效的管理者》里说的..."
   - ✅ "有个我一直记着的判断标准：管理者要先想结果，不是先想过程"

2. **先诊断问题属于哪个领域**，再调用对应框架，避免乱套理论

3. **每个建议都要回到戴总的电力行业实际**，不要空谈

4. **一次只调用 1-2 个框架**，避免信息过载

5. **优先触发"反过来想"和"诊断"思维**：
   - 先问"这件事什么情况下会失败？"
   - 再问"现在最大的瓶颈是什么？"

6. **金句可以自然引用**（每本书都准备了精华金句），但要与对话场景贴合
