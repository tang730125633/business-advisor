# 《苏世民：我的经验与教训》- 史蒂芬·施瓦茨曼
> 分类：商业思维 | 关键词：Think big、追求卓越、不要赔钱、10分人才、勿失良机、抓住机遇、精英管理、企业文化

## 一、核心思想

黑石集团创始人苏世民浓缩50年华尔街生涯为一条核心信念：**做大事和做小事的难易程度一样，所以要选一个值得追求的宏伟目标**（Think big, think long）。卓越不是高概率事件的堆叠，而是抓住寥寥可数的大机遇时全力以赴。他用25条原则把"勇敢、诚信、追求极致、善待他人"这些看似朴素的品质结构化，成为可执行的企业和人生操作系统。

## 二、核心思维模型

### 模型1：25条工作和生活原则（精华版）

**目标与格局**
1. 做大事和做小事难度一样，要选值得追求的宏伟目标。
2. 优秀高管是后天磨砺的，永远向顶尖的人和组织学习。
3. 主动给敬佩的人写信、打电话求见。

**理解他人**
4. 分析别人的问题并提出帮助——这是打开所有关系的钥匙。
5. 每个企业是封闭的集成系统，要懂每个部分和它们的协作。
6. 信息是最重要的商业资产，永远对新事物开放。

**职业姿态**
7. 年轻时要选陡峭学习曲线的工作，不要为短期声望妥协。
8. 印象非常重要，重诺守时、真实诚信、准备充分。
9. 再聪明的人也解决不了所有问题，开诚布公的团队无往不利。
10. 困境中的人只看自己的问题，而解决之道往往在于解决别人的问题。

**信念与诚信**
11. 信念必须超越自我。
12. 永远黑白分明，诚信必须毋庸置疑。
13. 勇往直前——别人谨慎时接受风险，别人犹豫时果断行动。
14. 永远不要骄傲自满，组织比想象中更脆弱。

**销售与推动**
15. 极少有人在首次推介中完成销售，要一次又一次地坚定推销愿景。
16. 看到巨大变革机会时，不要疑虑别人为何没动——你可能真的看到了他们没看到的。

**抓住机会**
17. 成功就是抓住寥寥可数的机遇，统筹合适的人和资源全力以赴。
18. 时间会对所有交易造成负面影响，谈判要让所有人坐得足够久。

**风险纪律**
19. **不要赔钱！！！** 客观评估每个机会的风险。
20. 要在准备好时做决定，而不是在压力之下。
21. 忧虑是积极的心理活动，可以让人看清负面风险。
22. 失败是组织最好的老师，坦诚谈论失败才能学到新规则。

**用人与善待**
23. 尽可能雇用10分人才——他们无须指令就能发现问题、设计解决方案。
24. 相信一个人本质是好的，就在他需要时帮他。
25. 每个人都有梦想，尽所能帮助别人实现他们的目标。

### 模型2：10分人才观
- **8分人才**：任务执行者。
- **9分人才**：擅长执行 + 能制订一流策略，全是9分就能成功。
- **10分人才**：无须指令，主动发现问题、设计解决方案、把业务推向新方向，还能吸引其他10分人才。
- 用人的全部精力应该放在识别和吸引10分人才上。

### 模型3：创业三项基本测试
1. 设想必须足够宏大。
2. 产品/服务必须独一无二。
3. 时机必须正确。
三者缺一不可，这是避免陷入平庸创业陷阱的核验清单。

### 模型4：卓越 = 零失误
- 卓越的实用定义：一切做到100%，零失误。
- 投资人/客户的5%表现不佳就意味着巨大损失。
- 这不是完美主义，是对专业边界的敬畏。

### 模型5：十大管理原则（补充）
- 坚持成长要不断提问、预测、主动寻求变革。
- 企业文化要兼具大公司规模和小公司灵魂。
- 精英管理 + 追求卓越 + 保持开放 + 坚守诚信，并雇用同样信念的人。
- 打破边界，进军自己"无权进入"的领域。

## 三、决策触发场景

- **纠结要做一个"小而稳"的项目还是"大而难"的项目**
  → 应用原则1：做大做小难度一样
  → 既然一样难，为什么不选大的？选那个回报能匹配努力的。

- **重大投资/收购决策时有人施压要快速拍板**
  → 应用原则20：在准备好时决定，而非压力下
  → 话术："我需要更多时间考虑，想清楚了再回复你。"

- **核心岗位是用老熟人还是引进10分人才**
  → 应用原则23：只雇10分人才
  → 10分人才贵但值得，他们会带来更多10分人才，形成正循环。

- **市场恐慌/行业低谷时要不要收手**
  → 应用原则19 + 原则13：不要赔钱 + 勇往直前
  → 客观评估风险（不要赔钱），但一旦看到真正的机会，在别人恐惧时果断出手。

- **看到一个"太大、不可能"的机会，犹豫是否追**
  → 应用原则16：不要疑虑别人为何没动
  → 问题越严峻，竞争越有限，对解决者的回报越大。

- **企业文化松散、员工缺乏紧迫感**
  → 应用"卓越 = 零失误"和精英管理
  → 重新定义"什么是好"，把卓越量化到每一个业务流程。

- **遇到难谈判、对方强势**
  → 应用原则18：时间对所有交易都有负面影响
  → 让所有人坐够长时间，信息会自然暴露，格局会自然变化。

## 四、电力行业映射

- **宏伟目标的选择**：电力行业民营老板容易陷入"只做熟悉的工程项目"的舒适区。苏世民会问戴总：在你这个资源和经验基础上，你追求的最宏伟目标是什么？是做一个地方性工程商，还是一个区域性综合能源服务商，还是新能源+储能+虚拟电厂的整合平台？做大做小一样难，格局要往上拔。

- **不要赔钱的铁律**：电力投资动辄几亿几十亿，重资产、长周期。苏世民的第19条是黑石的命根。电力行业同样适用：每一个大项目都要用"最坏情境"压力测试，政策、电价、利率、融资、技术路线任一环境变化下会不会伤筋动骨。

- **10分人才 = 电力行业最稀缺资源**：既懂传统发输配电、又懂新能源、又懂数字化、又懂资本运作的复合型高管极其稀缺。戴总应该把30%的时间花在识别、招募和保留10分人才上，而不是自己开所有会。

- **精英管理与企业文化**：电力行业国企味浓，民营企业也容易陷入"老臣政治"。建立精英管理机制——唯业绩、唯贡献、唯价值观——是跨越瓶颈的唯一路径。

- **危机中的勇往直前**：电力行业每隔几年就有政策周期/价格周期。别人恐惧时加杠杆收购优质资产，是黑石在2008年金融危机后大发横财的核心动作。这正是戴总在行业寒冬中应该思考的姿势。
