# 《价值》- 张磊
> 分类：商业思维 | 关键词：长期主义、动态护城河、时间的朋友、第一性原理、伟大格局观、选人用人、正和游戏、价值创造

## 一、核心思想

张磊以高瓴资本二十年实践，提炼出"长期主义"的底层世界观：把时间和信念投入能够长期产生价值的事情，拒绝零和博弈，与拥有伟大格局观的创业者同行。真正的投资只有一个标准——是否在创造对社会整体繁荣有益的真实价值。时间是最好的筛子，既筛掉虚假护城河，也筛出伟大的人与事。

## 二、核心思维模型

### 模型1：长期主义（Think Long）
- 本质是一种价值观，而非策略或方法论。
- "流水不争先，争的是滔滔不绝"——着眼长远，躬耕价值。
- 长期主义让人建立理性认知框架，不受短期诱惑和噪音影响。
- 在危机时刻，长期主义者看当下也相信未来，反而更敢于加注。

### 模型2：动态护城河（唯一真正的护城河）
- 张磊对巴菲特理论做了关键升级：世界上只有一条护城河，就是企业家持续疯狂地创造长期价值。
- 品牌、渠道、技术规模、知识产权等都有生命周期，是"静态护城河"，终将被时代冲刷。
- 真正的护城河是以用户为中心、不断自我颠覆、从第一曲线跳到第二曲线的能力。
- 企业如竹，要不断长出新的"竹节"（亚马逊、腾讯、字节跳动的演化史即证明）。
- 警惕政策保护型护城河——最脆弱。

### 模型3：三维决策框架——人、生意、环境、组织
- **看人**：看内心操守、价值追求、是否聚人、是否财散人聚、是否懂自己的边界。"看人就是在做最大的风控，比财务风控更重要。"
- **看生意**：看本质属性、解决的本质需求、演变可能、能否随时间积累复利。
- **看环境**：看政策、监管、供给、需求、人口结构、周期，寻找更长远、权重更大的外生因素。
- **看组织**：看组织能否承载动态创新。

### 模型4：第一性原理
- 不依赖类比和惯例，从事物本源出发思考。
- 用第一性原理重构商业逻辑，而不是在别人的框架里做小优化。

### 模型5：伟大格局观者的人才观
- 一流人才做三流生意可能做成一流；三流人才做一流生意也能把好牌打烂。
- 找"疯狂创造长期价值"的人，而非追热点的人。
- 成长型人才培养机制：教育和人才是永不退出的投资。

## 三、决策触发场景

- **要不要做一笔长周期、重资本的投资**（电网扩建、储能布局、新能源电站）
  → 应用长期主义 + 动态护城河
  → 不看5年回本周期焦虑，看这项投资是否在"疯狂创造真实价值"、是否能在20年维度建立动态壁垒。

- **短期KPI压力逼迫放弃战略投入**
  → 应用"做时间的朋友"
  → 区分哪些是"眼前的苟且"必须应对的，哪些是"诗和远方"不能动摇的。危机中依然坚守长期项目反而拉开差距。

- **识人、用人、给关键岗位**
  → 应用"看人即最大风控"
  → 不只看履历和能力，更看内心操守、边界感、财散人聚的气度、是否目光长远。

- **原有业务的护城河正在失效（如传统火电面临新能源冲击）**
  → 应用动态护城河
  → 承认旧护城河有生命周期，主动启动第二曲线（储能、综合能源服务、智慧电网），不要抱残守缺。

- **面对新兴行业不确定、看不清是否该进入**
  → 应用第一性原理
  → 剥离类比和噪音，回到"这件事是否在创造真实价值、解决本质需求"。

- **选择合作伙伴或关键合作客户**
  → 应用三维框架
  → 不只评估商务条件，同时评估人、生意本质、环境周期、组织能力。

## 四、电力行业映射

- **电力行业是典型的长期主义行业**：发电厂30年折旧、电网资产50年寿命。这个行业的DNA与长期主义高度契合。戴总所处的正是最应该拒绝短期主义的行业——用短周期的互联网逻辑看电力，必然做错决策。反过来，以长期主义为底层，反而能在同行焦虑时沉稳加注关键资产。

- **传统发输配电的动态护城河重构**：规模、牌照、电网物理资产曾是核心壁垒，但新能源、分布式、储能、虚拟电厂、电力现货市场正在瓦解部分老壁垒。真正的新护城河是"以用户侧为中心、持续创造综合能源价值的能力"——从卖电到卖服务、卖解决方案、卖可靠性。

- **人才梯队与接班人**：电力行业老龄化严重，技术断层风险高。应以"选人即最大风控"的标准，用十年维度培养既懂电力又懂数字化、新能源的复合型人才。对关键技术骨干，要"财散人聚"。

- **政策周期与战略定力**：双碳、电改、电价机制都在变。长期主义不是对抗政策，而是识别哪些变化是永恒趋势（清洁化、市场化、数字化），哪些是短期波动，在噪音中守住方向。
